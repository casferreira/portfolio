Projetos

TMDB

Projeto que teve como objetivo criar um catálogo de filmes. A página inicial possui a funcionalidade de paginação. Ao clicar no cartaz, o usuário é direcionado aos detalhes da obra, com informações como sinopse, nota de avaliação, elenco e filmes semelhantes.
A tecnologia utilizada foi React, com o consumo de uma API por meio da lib Axios; A transição de páginas foi feita por meio da lib React-Router e a estilização por meio de CSS e styled-components.

Labeninja

Projeto que teve como objetivo criar uma plataforma semelhantes ao GetNinjas. Na página é possível contratar ou se cadastrar como ninja. Ao selecionar um serviço, o usuário pode ver seus detalhes e adicioná-lo ao carrinho para posteriormente finalizar a compra. A página de serviços possui filtros, campo de busca e ordenação.
A tecnologia utilizada foi React, com o consumo de uma API por meio da lib Axios e a estilização por meio de CSS e styled-components.

Pokedex

O objetivo desse projeto foi criar uma pokedex. Na página é possível ver a lista de pokemons, sendo possível clicar no card para ver os detalhes e adicionar na pokedex. Há também a possibilidade de remover o pokemon, caso deseje.
A tecnologia utilizada foi React, com o consumo da Poke API por meio da lib Axios e a estilização por meio de CSS e styled-components.
